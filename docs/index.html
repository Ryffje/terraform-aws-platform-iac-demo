Terraform AWS Platform IaC Demo

Technical showcase of an AWS infrastructure provisioned with Terraform: VPC, Public Subnet, Security Group and EC2 with NGINX, plus secure access via SSH (restricted) and AWS SSM.

This page repository Main project repository

What this project demonstrates

  • Infrastructure provisioning with Terraform (VPC, Subnet, IGW, Route Table, SG, EC2)
  • Secure access patterns: SSH restricted + AWS Systems Manager (SSM)
  • Web server running on EC2 (NGINX / Port 80)
  • Visual evidence of the AWS resources and CLI access

Architecture diagram (AWS Icons)

Run it

  • terraform init
  • terraform plan
  • terraform apply
  • terraform destroy

Infrastructure lifecycle fully managed via Terraform.
