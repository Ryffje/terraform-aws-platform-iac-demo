Qué incluye

  • VPC (10.10.0.0/16)
  • Public Subnet (10.10.1.0/24)
  • Internet Gateway + Route Table (0.0.0.0/0)
  • Security Group: HTTP 80, SSH 22 (restringido)
  • EC2 (Amazon Linux) + NGINX + SSM Agent

Cómo correr

  • terraform init
  • terraform plan
  • terraform apply
  • terraform destroy

Tip: guardá variables en terraform.tfvars y documentalas en README.
